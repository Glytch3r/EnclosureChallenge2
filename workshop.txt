title=Enclosure Challenge
description=
description=[b]Enclosure Challenge 2[/b]
description=
description=Survive within a restricted cell-sized zone called "Enclosures".
description=
description=A challenge mod for Project Zomboid that forces the player to remain inside an Enclosure. Leaving the enclosure results in an immediate teleport back to the center.
description=
description=
description=[b]Modes:[/b]
description=Remote Mode teleports to a random enclosure must survive the assigned duration in order to win, it also conquers the enclosure
description=
description=Additive Mode is a continuously progressing survival game mode, Players may unlock multiple enclosures. this will only end if you quit or die.
description=You will gain unlock points everytime you win.
description=
description=
description=Mouse markers lets you look at enclosure properties
description=GUI shows you current enclosures properties
description=
description=The mod will show markers and map symbols automatically. This is yet Another feature that was added
description=
description=
description=[b]Features:[/b]
description=[list]
description=[*][b]Enclosure Restriction[/b] You're confined to a specific area. Stepping outside the cell will teleport you back.
description=[*][b]Timer-Based Challenge[/b] Survive for a set number of in-game hours to win (default: 168 hours / 7 days).
description=[*][b]Custom Rewards[/b] On completion, receive loot from a configurable reward pool.
description=[*][b]Edge Markers[/b] Visual boundary indicators can be toggled for clarity.
description=[*][b]Sandbox Support[/b] Tweak challenge duration, reward chance, and loot via Sandbox settings.
description=[*][b]Multiplayer Compatible[/b] Each player has independent challenge data. Players get invites when they're in the same Enclosure number.
description=[/list]
description=
description=[b]How to Start:[/b]
description=Use the context menu to trigger.
description=
description=[b]Sandbox Options[/b]
description=Configure how the Enclosure Challenge behaves in your world:
description=
description=[b]Default KeyBinds[/b]
description=Adjust GUI Opacity = O
description=Adjust GUI Position = P
description=Toggle MouseTip = N
description=
description=
description=[list]
description=[*][b]Allow Quit Challenge[/b] Allows players to manually quit the challenge at any time.
description=[*][b]Announce Challenge Status[/b] Shows challenge start/completion in chat. (Multiplayer only)
description=[*][b]Include Enclosure Number in Notifications[/b] Displays the enclosure number in public announcements.
description=[*][b]Challenge Hours[/b] Sets how long (in in-game hours) players must survive to complete the challenge.
description=
description=[*][b]Reward System[/b]
description=[list]
description=[*][b]List of Rewards[/b] Define reward items using item fulltype, separated by semicolons (e.g., Base.Katana;Base.Sledgehammer2).
description=[*][b]Reward Chance[/b] Set the chance (100%) for players to receive a reward after completing the challenge. Set to 0 to disable rewards.
description=[/list]
description=
description=Note: by default the reward items category is set to random
description=random guarantees 1 item to spawn from an item across all categories.
description=
description=[*][b]World Enclosure Limits[/b]
description=[list]
description=[*][b]Max World X[/b] Highest world X cell allowed for enclosures. Use to restrict challenge placement.
description=[*][b]Max World Y[/b] Highest world Y cell allowed for enclosures. Use to restrict challenge placement.
description=[/list]
description=[/list]
